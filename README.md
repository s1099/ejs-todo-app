# Todo App

Todo app built with [ejs](https://ejs.co/), Express and MongoDB

### Running

1. Clone the repo locally 
    ```
    git clone https://github.com/s1099/ejs-todo-app
    cd ejs-todo-app
    ``` 

2. Rename `.env.example` to `.env` and put in MongoDB URI

3. Install dependencies

    ```bash
    npm install
    # or
    pnpm install
    # or
    yarn install
    # or
    bun install
    ```

4. Start server
    ```bash
    npm main.js
    ```
